## ✨ Features

- 🔎 **Smart Product Search**
- 🛒 **Cart Management with LocalStorage**
- 📩 **Inquiry Modal for Direct Product Queries**
- 💳 **Smooth Checkout Flow**
- 🌙 **Dark / Light Mode Toggle**
- 🎯 **Responsive, Animated & UX-optimized UI**
